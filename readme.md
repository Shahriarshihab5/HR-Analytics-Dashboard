HR Intelligence & Attrition Analytics Dashboard
🎯 Executive Summary
This project delivers a comprehensive analytical overview of organizational workforce dynamics. By processing over 1,400 employee records, the dashboard provides actionable insights into attrition patterns, compensation structures, and employee satisfaction levels. The goal is to empower HR stakeholders with data-driven strategies to minimize turnover and optimize talent management.

🛠️ Technical Competencies
BI Visualization: Microsoft Power BI (Advanced DAX & Interactive Modeling)

Data Engineering: Power Query (ETL pipeline for data cleansing and normalization)

Database Architecture: Relational Schema Design using MySQL

Infrastructure: Scalable data modeling from flat-file (CSV) to Cloud-based SQL environments.

🏗️ Data Architecture & Scalability
A key highlight of this project is its scalability. While the front-end dashboard currently consumes data from a structured CSV for maximum portability, the backend logic is built upon a robust Relational Database Schema.

I have included a schema.sql file that defines the production-ready MySQL architecture, ensuring that the system can be seamlessly migrated to a live Cloud SQL instance (e.g., Aiven or AWS RDS) for real-time enterprise reporting.

📊 Strategic Business Insights
Attrition Correlation: Identified critical attrition triggers within specific Age Groups and Departments.

Retention Drivers: Analyzed the impact of "Monthly Income" and "Job Satisfaction" on long-term employee commitment.

Operational Analysis: Visualized the correlation between overtime, business travel, and workforce burnout.

📂 Repository Contents
HR_Analytics_Dashboard.pbix: Interactive Power BI reporting file.

HR_Analytics.csv: Cleaned and processed dataset.

schema.sql: MySQL schema definition for database migration.

Documentation: Implementation details and key findings.

🚀 Deployment & Usage
To interact with the dashboard:

Clone this repository.

Open the .pbix file using Power BI Desktop.

Use the slicers to filter data by Department, Education Field, or Marital Status for granular insights.
